``` markdown
# Основные возможности

- Тематические комнаты - публичные и приватные с системой приглашений

- Индикаторы активности - статусы "В сети", "Печатает", "Отошёл"

- Офлайн-сообщения - доставка сообщений при повторном подключении

- Реакции на сообщения - система лайков

- Упоминания (@username) - уведомления для упомянутых пользователей

- Защита от спама - ограничение длины и частоты сообщений
```
``` markdown
# Технологии

- Backend: Node.js, TypeScript, Express, Socket.IO

- Database: PostgreSQL с Prisma ORM

- Frontend: HTML, CSS, JavaScript
```
``` markdown
# Использование

- Выберите комнату из списка или создайте новую

- Настройте приватность при создании комнаты

- Общайтесь с поддержкой упоминаний и реакций

- Следите за статусами других участников
```
``` markdown
# Установка и запуск

- Установите зависимости:

| npm install

- Запустите сервер:

| cd server
| npx tsx watch index.ts

- Откройте файл client/index.html в браузере
```
